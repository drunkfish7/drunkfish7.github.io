产品经理小众工具推荐

推荐几个小众、非主流的产品经理小工具参考用：

收集资料

笔、本

笔和纸相对电子设备可以更自然地记录想法。笔推荐可以按动的笔，这样就不用担心笔帽找不到。推荐用本子代替纸，方便收纳。

Evernote

Evernote 很适合收纳各种信息，Evernote 有其他笔记应用没有的杀手锏功能，包括电脑端的 Web Clip 插件、手机端可以自动加强扫描文件、直接搜索图片中的文字等等。

时间管理

Evernote

推荐用 Evernote 作为 GTD+子弹笔记工具，淘汰其他时间管理工具，详阅《大脑减压的子弹笔记术》一书。

Diii

单核工作法可以帮助高效完成任务，Diii Notify 可以作为辅助工具，具体可以参考《单核工作法图解》。

文档输出

产品经理主要的输出产物包括「产品需求文档」等等，推荐 Markdown+在线文档的形式。文档中的文字、表格等用 Markdown 书写，流程图、原型图等用其他作图工具做。现在的很多在线文档都支持层级目录、协作及提交版本记录，可以方便协作、记录版本。

Sketch

Sketch 可以方便绘制界面图，可以将线框图（Wireframe）与说明、跳转关系放在一张图。另外，可以将界面可以复用的元素利用 Sketch 的「元件」功能形成组件库，保持设计样式统一、提高出图效率。

Typora

Typora 是一款 Markdown 编辑工具，可以实时预览。

OmniGraffle

OmniGraffle 是绘制流程图等图形的好工具，有很多模板可以选择。

MindNode

MindNode 可以方便输出思维导图。

uPic

uPic 可以将图片上传至图床，图床推荐 Github 图床。

Principle

苹果前员工写的《Creative Selection》一书中曾提到可交互原型（Prototype）在苹果开发过程中的重要性。Principle 可以方便制作交互动效原型。

Clickup 或 Thoughts

推荐使用在线 Markdown 文件输入各类文档，推荐 Thoughts（Teambition 推出，需付费）或 Clickup （近乎免费全功能），两者都支持 Markdown、支持目录层级、版本记录、协作、全终端。

伙伴云笔记

伙伴云笔记 可以用于管理需求的补充，按照不同维度分类需求，对需求的状态、优先级等标签化管理，支持协作。

项目协作

Clickup 或 Teambition

项目管理推荐使用 Teambition（需付费）或 Clickup （免费够用），两者都支持看板视图、日历视图。

群晖 NAS

国内公司好像用群晖的没有很多，群晖可以方便、安全地存储、共享团队文件。

Post-it 便利贴

Post-it 便利贴可以方便团队成员集思广益，还有 Post-it App 可以将便利贴快速转成电子版。

笔记模板

小弟整理了几个产品经理可能会用到的的几个模型的笔记模板，Evernote 格式，可以在这里查看或加入共享。
