让 macOS 更酷的应用推荐

macOS 是一个很酷的系统，有些应用可以让这个系统如虎添翼，下面是小弟的一些推荐：

Popclip

Popclip 将 iOS 的 edit menu 搬到了 Mac，当选中文本时，会自动显示菜单，比如复制、粘贴、搜索、拼写等等，更酷的是支持插件，很多高频、重复的操作都可以通过插件实现。

jitouch

jitouch 是一个手势增强应用，可以让 Magic Mouse 和 Magic Trackpad 更酷。应用可以拓展 Magic Mouse 和 Magic Trackpad 的多点触控手势，可以很方便地执行一些高频的操作，比如浏览器中切换标签页、关闭标签页、真最小化等等。小弟觉得这款比 BetterTouchTool 设置上更合理，内置了很多手势，开箱即用，且设置直觉、简单。

另外，超酷的一点是，应用支持全局鼠标手势（Mouse Gestures），类似于 Windows 上的 StrokeIt ，只要按住右键画一个字母就可以执行设定的操作。因为是全局手势，还可以与其他应用联合使用。

Moom

Moom 可以实现类似 Windows 的窗口大小快捷管理，安装后悬停「全屏」按钮会出现最大化等按钮，可以快速调整窗口大小，支持快捷键。

HyperDock

HyperDock 可以实现类似 Windows 7 任务栏窗口预览的功能， 在 Dock 栏上悬停应用图标就可以预览应用窗口，单击可以快速切换窗口。

另外建议使用这个应用时，在系统设置→ Dock 中开启「最小化窗口至应用图标」这个选项。

The Unarchiver

The Unarchiver 是一款解压应用，支持格式很多，可以直接在当前路径解压。若需要查看压缩包内文件，可以使用 The Archive Browser（同一个开发者开发，目前已下架，可以在这里下载）。

Paste

Paste 是一款高颜值的剪贴板增强应用，可以快速访问剪贴板的历史，支持搜索、以纯文本形式粘贴、合并粘贴等等。

Alfred

Alfred 可以增强 Spotlight ，快速搜索、浏览应用程序、文件等等，可以通过扩展实现自动化。

搜狗输入法

Mac 自带拼音输入法自定义部分太少，Mac 平台输入法选择也太少，搜狗输入法 是一个选择，支持辅助码，还自带一个英文输入法。

1Password

1Password 可以管理各种密码、地址、身份信息等等，可以通过 iCloud 同步。

MindNode

MindNode 是一款漂亮、好用的脑图工具，可以帮助理清思路、捕捉灵感。

CleanMyMac X

CleanMyMac X 可以帮助现在应用，清除一些不需要的文件，如緩存、日誌等等。

Mission Control Plus

Mission Control Plus 如其名，可以增强 Mission Control 的功能，可以直接在 Mission Control 中关闭窗口，支持自定义快捷键。

TotalSpaces2

TotalSpaces2 可以增强多桌面体验，可以自定义桌面快捷键、切换方式等等。

Itsycal

Itsycal 是一个菜单栏小工具，可以方便访问日历。

Network Speed Monitor

Network Speed Monitor 可以在任务栏显示网速，App Store 下载地址在这里。

Bartender

Bartender 可以增强菜单栏图标管理，可以方便地隐藏、重新排列等等，设置支持快捷键、搜索等等。

A Better Finder Rename

A Better Finder Rename 可以帮助快速重命名多文件。

Movist Pro

Movist Pro 是一个好用的媒体播放器，功能全面。

Dropshelf

Dropshelf 和 Popclip 是同一个开发者开发的，功能是将屏幕侧边变成一个文件临时放置点，可以方便拖动放一些临时文件。

Visual Studio Code

Visual Studio Code https://code.visualstudio.com 是微软出的代码编辑器，比一般文本编辑功能多，比 IDE 轻，简单文本编辑很合适。Visual Studio Code 支持插件，常用插件可以参考这里。

Tickeys

Tickeys 可以为敲击键盘增加音效，增加打字的感觉。

CaptuocrToy

CaptuocrToy 一个使用在线接口识别截图中文字的工具。


一个使用在线接口识别截图中文字的工具。

Downie

Downie 可以帮助下载网站视频，比如 Youtube 等等，支持多分辨率、字幕下载，内置一个浏览器方便下载，Facebook 粉丝专页在这里。

AirServer

AirServer 可以将 Mac 模拟成 Apple TV，方便将 iOS 设备投射到 Mac 上。
