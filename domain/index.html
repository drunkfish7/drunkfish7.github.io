Freenom 申请免费域名

我们有时候有需要用到域名，比如建博客、搭建伪装流量、访问家中的设备等等，如果不想花钱买域名，Freenom 提供了 .tk、.ml、 .ga、.cf、.gq 域名，是一个选择。

获取域名

操作时建议全程美国 IP。首先打开 Freenom 网站，默认注册是隐藏的，直接在搜索框中需要的域名，务必加后缀，如「drunk-fish.tk」，若显示绿色的「Selected」按钮，说明域名可用。依次输入想要的域名（需加后缀「.tk、.ml、 .ga、.cf、.gq」 ），之后点击「checkout」按钮，

选择「12个月」，

输入邮箱或使用 Google 或 Facebook 账号登录，登录邮箱打开验证链接，

填写个人信息，其中美国地址建议在这里生成，

此时申请域名成功。

注意：若出现以下提示建议更换美国 IP 重试。

域名解析

申请域名之后可以根据实际情况设置域名解析。首先找到要解析的域名，Freenom 可以选择「服务」，之后选择「我的域名」，

找到要解析的域名如 drunk-fish.tk，选择「管理域名」，

选择「管理 DNS 记录」，

之后就可以修改 DNS 解析，

其中 drunk-fish.ga 是一级域名， www.drunk-fish.ga 是二级域名， www.xiaoyu.drunk-fish.ga 是三级域名，二级、三级域名都是子域名。

主机记录是域名前缀，如 www.drunk-fish.ga， www 即为这个子域名的主机记录。其中主机记录常用的有以下两个：

  1. A 记录

如果虚拟主机是独立 IP，使用 A 记录类型，记录值填写服务器 IP。主机记录内容填写域名前缀，常见用法有:

  • www 解析至域名 www.drunk-fish.ga
  • @ 或留空 直接解析至主域名 drunk-fish.ga
  • * 解析至域名 *.drunk-fish.ga
  1. CNAME 记录

如果虚拟主机不是独立IP，使用 A 记录类型，用法与 A记录相似。

TTL 是一条域名解析记录在 DNS 服务器中的存留时间，一般默认值即可。

注意：域名解析需要一定时间，几分钟至几小时不等，可以在终端机输入 ping 域名检查是否解析成功。

域名续期

记得设好提醒续约域名，域名快到期也会收到邮件，记得续约。
