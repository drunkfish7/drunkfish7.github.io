Chrome 插件推荐

记得 08 年的小弟生日那天，晚上发现一款浏览器在这一天发布了公开测试版，名字是一个不认识的英文词语 Chrome，感觉和这款浏览器有源。

许多年过去了，Chrome 从小众成为用的人最多的浏览器，功能也越来越多。下面是小弟的一些插件推荐（除了 Chrome，也大部分适用于 Chromium 内核的浏览器，如 Opera、Vivaldi、Brave、Edge 等）：

插件推荐

以下 Chrome 插件均可在 Chrome 线上应用商店安装。

1Password

此插件需配合 1Password 程序使用，快速填入密码及表单、保存密码，支持快捷键，比 Chrome 自带密码管理功能更多。

安装地址

Evernote Web Clipper

可以快速将网页内容保存到 Evernote，还可以简单标注、加标签，支持快捷键。

安装地址

AdGuard AdBlocker

小弟用过好几款广告过滤插件，这款效果最好，AdGuard 本身主要的盈利方式是出售付费的 AdGuard 应用，有理由相信拦截效果不错。

安装地址

Tampermonkey

可以为浏览器安装脚本，脚本下载可以到这里。

安装地址

Context Menus

此插件可以增强右键，比如以图搜图等等，插件已多年未更新，已从 Chrome Store 下架，可以从这里离线下载安装。

Dark Reader

人们使用屏幕的时间越来越长，dark mode 越来越被人接受，这个插件可以页面实现 dark mode。

安装地址

Extension Manager

这款插件可以快速禁用、启用插件，还可以将插件分组。

安装地址

User-Agent Switcher for Chrome

Google 出的方便切换浏览器标识的小插件，可以方便查看不同设备上的网页展示效果。

安装地址

下载管理

这个插件可以实现快速查看下载文件列表。

安装地址

LastPass

另外一款密码管理插件，LastPass 免装额外应用，可以账号登录同步数据。小弟是个人的密码用 1Password，公司的密码用 LastPass。LastPass 的最快捷用法是 ⌘+L 定位至地址栏，之后输入「lp」空格就可以直接搜索密码。

安装地址

Next Page

这款插件可以实现类似 GreenBrowser 的自动翻页功能，提供快捷的翻页功能，免去寻找和点击翻页链接，适合在浏览搜索结果、论坛时使用。

安装地址

Workona

我们常常打开很多标签页，查找、管理很不方便。Workona 可以实现标签页分组，保存已打开的标签页，类似 GreenBrowser 群组功能。另外按下 ⌥+f 还可以实现类似 Spotlight 的功能，快速定位标签页。

安装地址

SetupVPN

SetupVPN 是一个免费 VPN 插件，注册登录之后有很多免费线路可以选择，可以做为一个 VPN 备选。

安装地址

Page Marker

用这个插件可以简单的对网页内容做标注，画个草图或示意图什么的，类似 GreenBrowser 的鼠标手势轨迹功能。

安装地址

购物党

购物党可以方便查看淘宝等购物网站历史价格、比价等等。

安装地址

Google Arts & Culture

这个插件可以实现每打开新标签页显示一幅艺术画，比其他新标签类插件酷多了，图的来源是 Google Arts & Culture。

安装地址

彩云小译

这个插件可以实现中英文对照翻译。

安装地址

Server IP

这个插件可以显示网址 IP 所在国家的国旗。

安装地址

Aerys - 窗口标签管理器

Aerys 可以分类显示标签页，拖动改变顺序，方便在窗口之间移动标签页、快速重新打开最近关闭的标签页。

安装地址

Clearly

这个插件可以实现类似 Safari 的阅读模式，侧边栏还可以显示文章的层级结构，唯一不足的是中文字体只能使用非衬线体。

安装地址

Double Click Closes Tab

这个插件可以实现双击或三击关闭标签页，注意，这个是双击页面任意位置关闭标签页，不是双击标签页。

安装地址

扩充功能/扩展管理器（Extension Manager）

这款插件可以方便管理插件，快速禁用、启用插件、分组等等。

安装地址

自带功能

快速指定搜索

Chrome 可以方便地快速指定搜索，在地址栏输入 chrome://settings/searchEngines 回车，可以看到浏览的网页中含有搜索引擎的，可以指定快捷键，如指定 「yt」为 YouTube 搜索，之后直接在地址栏输入「yt」空格后面加关键词可以直接在 YouTube 搜索。

开启同步功能

Chrome 同步功能很完善，可以同步书签、历史记录、插件等等，推荐开启。另外可以在「我的 Google 活动」中看到历史记录。

设置插件快捷键

地址栏输入 chrome://extensions/shortcuts 回车，可以对插件设置快捷键。

几个 chrome:// 设置

chrome:// 可以对 Chrome 进行很多设置，以下几个推荐打开：

  • 打开阅读模式

Chrome 自带阅读模式，需要手动打开，在地址栏输入 chrome://flags/#enable-reader-mode ，选择「Enable」，之后重启浏览器。

  • 开启多线程下载

地址栏输入 chrome://flags/#enable-parallel-downloading 回车，选择「Enable」重启，可以打开多线程下载，提高下载文件速度。

  • 打开悬停标签页显示

    Chrome 自带悬停标签页显示，打开方式：地址栏输入 chrome://flags/#tab-hover-cards，可以选择「Enable B」或「Enable C」。

  • 打开自带标签页分组显示（v83 以上版本）

    地址栏输入 chrome://flags 找到「tab groups」选择「Enabled」。

Chromecast 功能

可以在 Chrome 浏览器内将网页视频、音频投放至 Chromecast 设备。

网页翻译

依托自家的 Google Translate，Chrome 可以提供比其他浏览器更无缝的网页翻译。
