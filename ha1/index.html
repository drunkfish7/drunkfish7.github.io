Home Assistant 是最酷的开源智能家居平台，可以很方便地接入各种设备、DIY，以及 Homekit、Google Assistant、Alexa。

运行 Home Assistant 可以选择树莓派，或者其他 Linux 硬件，下面是群晖 Docker 的安装步骤：

在群晖套件中心搜索「docker」安装 Docker（图标是小鲸鱼顶集装箱） 并打开，

在「注册表」标签中搜索「home assistant」，下载 Home Assitant 的映像，

等待映像下载完成，完成后群晖会有通知。在「映像」标签中找到下载的映像双击，

选择「高级设置」，

在「高级设置」标签中根据偏好选择是否「自启动」，

在「卷」标签中添加一个文件夹，这个文件夹将存放 Home Assistant 的配置文件，挂载路径为 /config，

在「网络」标签中选择「与宿主网络相同」，

在环境中添加 variable = TZ 和 value = Asia/Shanghai ，

确认「高级设置」，点击「下一步」，之后「应用」，在浏览器访问 http://群晖 IP:8123 。至此，Home Assistant 安装成功。
